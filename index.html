准备工作：确保能科学上网

Codex 用的是 OpenAI 的服务，国内网络直接连不上。你需要先准备好科学上网工具，确保能正常访问 openai.com 和 codex.openai.com。这一步搞定，后面就是一马平川。

⚠️ 注意：科学上网是使用 Codex 的硬性前提，没有这一步后面的都白搭。推荐 neworld.space，稳定好用。
1

去 Codex 官网下载

浏览器打开 codex.openai.com（记得开着科学上网），点首页的"Download"按钮。Windows 选 .exe 版本，Mac 选 .dmg 版本。

2

安装并登录

双击下载的文件安装。装好打开 Codex，会提示登录。用你的 OpenAI 账号登录就行。没有账号的话在官网免费注册一个。

💡 注册 OpenAI 账号需要国外手机号验证，这一步也需要科学上网。搞不定的话加微信我帮你。
3

验证安装成功

打开电脑终端（Windows：按 Win+R，输入 cmd 回车），输入：

codex --version

如果显示版本号（比如 v1.x.x），说明装好了。之后每次使用记得保持科学上网开着就行。

使用教程

怎么让 Codex 帮你干活

打开 Codex 你会看到一个聊天窗口。就像跟朋友聊微信一样，用中文把你要做的事情说给它。

⚠️ 每次使用前，确保科学上网是开着的。 Codex 的所有功能都依赖 OpenAI 服务器，网络不通就动不了。

示例 1：帮我做个个人网站

直接输入："帮我做一个个人博客网站，要有首页、文章列表、关于我三个页面，风格简洁大方。" Codex 会自动建项目文件夹、写代码、还能帮你预览效果。

示例 2：帮我处理表格数据

把 Excel 文件拖进对话框，然后说："把这个表格里销售额超过 10 万的行挑出来，按销售额从高到低排，生成新表。" Codex 读完文件、写脚本处理、输出结果。

示例 3：帮我改 Bug

在你的项目文件夹里打开 Codex，把报错信息贴进去："运行 npm start 报了这个错，帮我修一下。" Codex 会读你的代码、定位问题、修复、再跑一遍确认。

示例 4：帮我写自动化脚本

比如："帮我写个脚本，每天早上 9 点自动从指定网站抓取新闻标题，存成文本文件。" Codex 写脚本、配定时任务、告诉你放哪个文件夹。

核心思路：说人话讲清楚你要什么，剩下的交给 Codex。 说不清楚也没事，它会主动问你，聊几轮就把活干完了。
